# Knoweledge Graph Verbalizer

# Setup

In order to setup the environment locally you must use to terminals, one to set up the website and one to setup the server.

# Website Setup

Open a terminal at the root folder of the repo.
`cd website`

`yarn install`

`yarn start`

# Server Setup

Open a terminal at the root folder of the repo.
`cd rosae-server`

`npm install`

`npm start`
